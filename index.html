Mis Proyectos

Animanga Universe - Aplicación Android

Una aplicación para gestionar colecciones de anime, similar a MyAnimeList. Permite agregar, clasificar y calificar anime con una interfaz amigable. Tiene un buscador para poder buscar y calificar las obras ya vistas o descubrir obras nuveas. También tiene un foro para poder compartir opiniones y debatir sobre las obras con la gente de la comunidad

Tecnologías: Java, Android Studio
